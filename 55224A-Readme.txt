*****   70-475 Exam preparation   *****
All the exam objectives for 70-475 are covered in 552241A & 552242A.  
552241A covers the first two exam objectives.  Designing big data batch processing and interactive solutions / Designing big data real-time processing solutions
552242A covers the third objective.  Operationalize end-to-end cloud analytics solutions


*****   Class Delivery   ***** 
55224A has a modular course design allowing the lab exercises to be taught in any order the trainer chooses.  This also allows customized deliveries of the material in combination with other Azure, PowerShell or SQL Server classes.  
For additional help in configuring such deliveries or choosing the right setup strategy, please read the Trainer Preparation Guide (55224A-ENU_TrainerPreparationandNotes.pdf) or contact the author (neil@neiltucker.biz).


*****   Setup Files   *****
All instructor and setup files are on the Student CD included with the course manual.  If you do not have a hard copy of the manual, these files can be downloaded from Skillpipe using the following directions:
1. Purchase a digital copy of the course from the Courseware Marketplace web-site (Free for MCTs)
2. From the Courseware Marketplace web-site, click on the Skillpipe link and login to the web-site (http://skillpipe.courseware-marketplace.com)
3. Click on the option to "read the course online"
4. Click on the option for "Book Info & Additional Files"
5. Download all the files in the list presented to create your own Student CD


*****   55224A Course Updates   *****
Microsoft Azure is a great source of cloud utilities and resources that are constantly being improved and updated.  To keep pace with these changes, 55224A labs will be tested regularly and updated when appropriate to ensure the exercises stay up-to-date.  These updates will be made available to past and present students.


*****   55224A Exam Objectives   *****
Microsoft has changed the original four exam objectives to be three.  Some of the material from the original "Design Machine Learning Solutions" have been rolled over into the third objective "Operationalize end-to-end cloud analytics solutions".
To keep up-to-date with the latest changes for exam 70-475, do a web search for "Exam 70-475" and review the information on the Microsoft Learning web-site.
